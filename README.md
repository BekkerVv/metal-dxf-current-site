# Парк Металл — DXF калькулятор v0.4

Локальный браузерный калькулятор заказа: DXF, позиции, расчет, КП, производство и раскладки.

## Запуск

1. Распаковать архив.
2. Открыть папку в VS Code.
3. Запустить `index.html` через Live Server.

Можно также открыть `index.html` двойным кликом, но через Live Server стабильнее.

## Что нового в v0.4

- Интерфейс разбит на вкладки: Заказ, Позиции, Расчет, КП, Производство, Раскладки, База цен, Настройки.
- База цен убрана в отдельную вкладку.
- Минималка резки выключена по умолчанию.
- КП: одна деталь = одна строка, толщина отдельной колонкой.
- Производственный отчет: таблица с маленьким превью детали.
- Внутри позиции появились поля: RAL, текстура, сварка ₽/шт, доп. услуги, примечание в производство.
- Услуги входят в цену детали: резка, гибка, полимерка, сварка, доп. продажа.
- Раскладки: можно прикреплять JPEG/PNG, указывать материал, толщину, лист, расход, заполнение.
- Печать / PDF для КП и производственного отчета через печать браузера.
- Проект JSON сохраняет позиции, цены, услуги, КП/производственные поля и раскладки.

## Важно

DXF-файлы не отправляются в интернет. Расчет идет в браузере.

Производственный отчет не показывает цены — только техническую информацию.

Минималку заказа добавлять вручную через “Дополнительные строки заказа”.
