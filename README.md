# Парк Металл — DXF калькулятор v0.6

Рабочая версия после правок Владимира.

## Главное в v0.6

- Убран визуальный мусор: карточки, скругления, лишние обводки и крупные поля.
- Интерфейс ближе к Excel: плотные таблицы, ячейки, компактные кнопки.
- В верхней панели оставлены только: сохранить, загрузить, скопировать КП, очистить.
- Слева добавлен бургер-меню:
  - открыть проект;
  - сохранить проект;
  - история последних сохранений;
  - визуальные настройки;
  - подсказка по программным настройкам.
- Визуальные настройки:
  - светлая/темная тема;
  - акцентный цвет;
  - размер превью;
  - показать/скрыть подсказки.
- Себестоимость перенесена на уровень всего заказа во вкладке «Расчет»:
  - металл: количество × цена;
  - доп. себестоимость: сумма вручную;
  - итоговая себестоимость;
  - маржа ₽ и %.
- По позициям себестоимость больше не считается.
- В расчетной таблице остались коэффициенты и суммы продажи по позициям.
- Раскладки:
  - JPEG/PNG открываются как визуальный просмотр;
  - клик по превью открывает окно увеличения;
  - LXDS можно добавить в список, но автоматическое JPEG-превью требует локального сервера/конвертера.

## Запуск

1. Распаковать архив.
2. Открыть папку в VS Code.
3. Открыть `index.html` через Live Server.
4. Внизу страницы должна быть версия `v0.6 current site`.

## Важно

Проекты по-прежнему сохраняются файлом `.json`, который нужно класть в папку заказа.
История внутри сайта хранит только краткую информацию о последних сохранениях.
