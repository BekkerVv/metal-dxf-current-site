# Парк Металл — DXF калькулятор current site v0.2

Локальная браузерная версия. Работает без интернета: DXF читаются прямо в браузере.

## Запуск

1. Распаковать архив.
2. Открыть папку `metal_dxf_current_site_v0_2`.
3. Запустить `run.bat`.
4. Откроется `http://127.0.0.1:5180`.

Можно также просто открыть `index.html`, но через `run.bat` стабильнее.

## Что умеет v0.2

- загрузка одного/нескольких DXF;
- загрузка папки с DXF;
- ручные позиции без DXF;
- расчет длины реза, врезок, габарита;
- предварительный вес по габариту;
- `+толщина` к габариту заготовки;
- металл, резка, гибка, полимерка;
- расход порошка `0,15 кг/м²`;
- себестоимость и продажная цена;
- НДС и маржа;
- дополнительные строки: сварка подрядчика, метизы, упаковка и т.п.;
- превью DXF;
- игнорирование слоев по словам;
- предупреждения по сплайнам, эллипсам, двойным линиям и незамкнутым контурам;
- объединение похожих/одинаковых деталей;
- сохранение/загрузка проекта JSON;
- экспорт CSV;
- копирование краткого КП.

## Важное

Вес металла считается предварительно по габариту детали/развертки, а не по фактической раскладке CypCut.
Финальный расход металла после раскладки менеджер корректирует отдельно.
