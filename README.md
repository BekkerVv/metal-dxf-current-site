# Парк Металл — DXF калькулятор v1.1

Версия v1.1 — обновлена база цен по материалам, резке, прожигу и гибке по данным от 10.06.2026.

## Что добавлено в v1.1

- Добавлена база материалов:
  - оцинковка / `оц`;
  - `08пс хк`;
  - `гк ст3`;
  - `09Г2С`;
  - сохранены нержавейка и алюминий.
- Для материалов используется строгая проверка по прайсу: если материала, толщины или раскроя нет — строка/расчет подсвечивается ошибкой.
- В себестоимости заказа цена металла берется из прайса автоматически по:
  - материалу;
  - ширине листа;
  - длине листа;
  - толщине;
  - количеству листов.
- Для `гк ст3` учтен разный прайс на 3 мм:
  - 1250×2500;
  - 1500×6000.
- Добавлены группы резки:
  - черный металл;
  - оцинковка;
  - нержавейка/алюминий.
- Резка считает не только цену за метр, но и прожиг по прайсу для выбранной группы материала.
- Гибка теперь зависит от толщины и длины гиба:
  - до 500 мм;
  - до 1500 мм;
  - до 3000 мм.

## Важно

Цены металла внесены по колонке `цена+4`, потому что она уже дана отдельно в твоем прайсе. Если надо использовать чистую `Цена р/кг`, ее можно импортировать/поправить во вкладке «База цен».

## Установка

1. Скопировать файлы из архива в папку проекта с заменой:
   - `index.html`
   - `app.js`
   - `styles.css`
   - `README.md`
   - `run.bat`
2. Открыть через Live Server.
3. Внизу должно быть `v1.1 current site`.

## Коммит

```text
update price database v1.1
```
