# Парк Металл — DXF калькулятор v0.5

Локальный браузерный калькулятор заказа: DXF, позиции, финальный расчет, КП, производственный отчет и раскладки.

## Запуск

1. Распаковать архив.
2. Открыть папку в VS Code.
3. Запустить `index.html` через Live Server или `run.bat`.
4. Открыть сайт в браузере.

## Что нового в v0.5

- Вкладка **Позиции** стала основной рабочей таблицей.
- Основные поля редактируются прямо в таблице:
  - количество;
  - материал;
  - толщина;
  - резка;
  - количество гибов;
  - длина гиба;
  - полимерка 0/1/2 слоя;
  - RAL;
  - текстура;
  - сварка ₽/шт;
  - доп. услуга;
  - примечание.
- Добавлено маленькое превью детали в строке.
- Большой редактор позиции убран с рабочего экрана.
- Добавлено быстрое заполнение:
  - применить значение к выбранным строкам;
  - применить значение от выбранной строки вниз.
- Вкладка **Расчет** стала местом финальной подгонки цены:
  - коэффициент металла по позиции;
  - коэффициент резки по позиции;
  - коэффициент гибки по позиции;
  - коэффициент полимерки по позиции;
  - фактический металл, кг;
  - себестоимость металла;
  - себестоимость резки;
  - себестоимость гибки;
  - себестоимость краски/полимерки;
  - себестоимость сварки;
  - себестоимость доп. услуг.
- Итоги выводятся под колонками расчета.
- Себестоимость больше не берется автоматически из продажных коэффициентов. Она считается из фактически введенных данных во вкладке **Расчет**.

## Логика работы

1. Загрузить DXF.
2. В **Позициях** быстро заполнить количество, материал, толщину, гибку, полимерку и услуги.
3. Сделать раскладку.
4. В **Расчете** внести фактическую себестоимость и коэффициентами подогнать продажную цену.
5. Перейти в **КП** и распечатать PDF.
6. Перейти в **Производство** и распечатать технический отчет.

## Важно

Все расчеты выполняются локально в браузере. Файлы не отправляются в интернет.
