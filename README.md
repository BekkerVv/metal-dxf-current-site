# Парк Металл — DXF калькулятор v1.4

Версия v1.4 добавляет вкладку **ИИ**.

## Обычный запуск без ИИ

```bat
run.bat
```

Откроется локальный сайт. Простые команды во вкладке ИИ будут работать через локальный разбор, без API.

## Запуск с настоящим ИИ

1. Установи Node.js, если его нет.
2. Скопируй `.env.example` в `.env`.
3. В `.env` вставь ключ:

```env
OPENAI_API_KEY=sk-...
OPENAI_MODEL=gpt-5.5
PORT=5180
```

4. Запусти:

```bat
run_ai.bat
```

5. Открой:

```text
http://127.0.0.1:5180
```

Ключ OpenAI не вставляется в `index.html` и не хранится в браузере. Его читает только локальный `server.js`.

## Что умеет вкладка ИИ

ИИ получает краткий JSON текущего проекта и может предложить действия. Действия применяются только после нажатия **Применить действия**.

Примеры:

```text
проверь заказ
всем материал 08пс хк
выбранным толщина 2 мм
всем RAL 9005 1 слой
гибка 2, длина гиба 500
добавь доставку 5000
себестоимость гк ст3 5 мм 1500х6000 1
```

## Безопасность

ИИ не меняет заказ сам. Он только готовит список действий. Менеджер проверяет список и нажимает **Применить действия**.
