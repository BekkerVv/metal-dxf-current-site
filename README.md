# Парк Металл — DXF калькулятор заказа v1.2

Локальный браузерный калькулятор для DXF, КП, производственного отчета и предварительного расчета заказа.

## Что изменено в v1.2

- Себестоимость заказа по умолчанию больше не заполнена автоматически.
- `state.orderCost.sheets = 0`, пока количество листов/заготовок не введено — себестоимость и маржа показываются как `не заполнено`.
- Себестоимость металла считается по листу: материал × ширина × длина × толщина × количество × цена ₽/кг.
- Добавлен компактный редактор геометрии выбранной позиции:
  - габарит X;
  - габарит Y;
  - длина реза;
  - врезки;
  - площадь покраски.
- Для DXF можно вернуть геометрию из исходного файла кнопкой `Вернуть из DXF`.
- Если гибы больше 0, а длина гиба равна 0 — строка получает замечание.
- В базе цен добавлена вкладка `Раскрои листа`:
  - материал;
  - толщина;
  - ширина;
  - длина;
  - цена ₽/кг.
- Поля себестоимости заказа обновляют расчет по `change`, а не на каждый символ, чтобы таблица не дергалась при вводе.

## Запуск

1. Распаковать архив.
2. Открыть папку в VS Code.
3. Запустить через Live Server или `run.bat`.
4. Открыть `index.html`.

## Важно

Расчет работает локально в браузере. DXF и проекты не отправляются в интернет.
