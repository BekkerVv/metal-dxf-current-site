# Парк Металл — DXF калькулятор v0.8

Версия v0.8 — стабилизация после теста v0.7.

## Что исправлено

- Починены кнопки печати КП и производственного отчета.
- Починена загрузка JPEG/PNG раскладки.
- Маржа больше не показывает 100%, если себестоимость заказа не заполнена.
- Себестоимость считается только на весь заказ: металл + доп. себестоимость.
- Исправлена ошибка `MATERIAL_DENSITY is not defined`.
- Парсер понимает формат `Лист 12 400 х 400 — 6 шт` и берет толщину 12 мм.
- Ложные предупреждения по прямоугольнику из четырех LINE уменьшены: замкнутый контур из линий не считается как 4 незамкнутые линии.
- Цены по металлу/резке/гибке берутся строго по прайсу. Если материала или толщины нет в базе цен, строка выделяется ошибкой.
- Верхние лишние кнопки скрыты: ручная позиция, объединение дублей, CSV.
- Колонка `Доп.` переименована в `Доп. прим.`, чтобы было понятно: это текст, не деньги.

## Установка

1. Скопировать файлы из архива в папку проекта с заменой:
   - `index.html`
   - `app.js`
   - `styles.css`
   - `README.md`
   - `run.bat`
2. Открыть через Live Server.
3. Внизу должно быть `v0.8 current site`.

## Коммит

```text
stabilize calc and price validation v0.8
```
