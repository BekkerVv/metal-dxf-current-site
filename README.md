# Парк Металл — DXF калькулятор v1.5

Версия v1.5 переводит сайт на локальный сервер через `run.bat`, чтобы можно было сохранять проект прямо в папку заказа.

## Как запускать

1. Распаковать архив.
2. Открыть папку.
3. Запустить `run.bat`.
4. Браузер откроет:

```text
http://127.0.0.1:5180
```

Если браузер не открылся сам — скопируй адрес из окна.

## Куда сохраняет

По умолчанию сервер использует папку:

```text
C:\Users\<пользователь>\Desktop\Заявки Новая форма
```

Можно поменять корень через переменную окружения `ORDERS_ROOT` перед запуском.

## Новый сценарий сохранения

1. Вставить строку заказа:

```text
7174_07.07.2026_Бетомас
```

2. Нажать `Разобрать`.
3. Заполнить заказ.
4. Нажать `Сохранить` или `Сохранить как`.
5. Выбрать/ввести папку заказа.

В папке появятся:

```text
7174_07.07.2026_Бетомас.project.json
Открыть заказ.bat
Открыть заказ.url
```

`Открыть заказ.bat` открывает сайт сразу с этим проектом, если локальный сервер запущен.

## Важно

- Это не OpenAI и не платный API.
- Все работает локально на ПК.
- Проект не перезаписывает базу цен сайта.
- Если сервер не запущен, кнопка сохранения сработает как обычное скачивание JSON.
