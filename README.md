# Парк Металл — DXF калькулятор current site v0.3

Локальная браузерная версия DXF-калькулятора.

## Запуск

Через VS Code:

1. Открыть папку проекта в VS Code.
2. Открыть `index.html`.
3. Правой кнопкой → `Open with Live Server`.

## Что добавлено в v0.3

- Блок **База цен** прямо в интерфейсе.
- Таблица цен металла по материалу и толщине.
- Таблица цен резки по толщине.
- Таблица цен гибки по толщине.
- Настройки покраски: ₽/м², себестоимость, расход порошка, цена порошка, минималка.
- Импорт/экспорт базы цен JSON.
- Автоподстановка цены по ближайшей толщине.
- Редактирование материала/толщины в позиции с пересчетом.

## Важно

Вес металла считается предварительно по габариту детали/развертки, а не по фактической раскладке CypCut.
Финальный расход металла после раскладки менеджер корректирует отдельно.
